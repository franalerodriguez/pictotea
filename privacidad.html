Política de privacidad de PictoTEA

Última actualización: 12 de marzo de 2026

Esta política de privacidad describe cómo la aplicación PictoTEA gestiona la información generada durante el uso de la aplicación.

URL oficial de esta política:

https://franalerodriguez.github.io/pictotea/privacidad.html

1. Datos recopilados

PictoTEA no recopila datos personales identificables como nombre, correo electrónico, dirección o ubicación del usuario.

La aplicación puede almacenar localmente en el dispositivo información relacionada con el uso, como:

  • Frases construidas con pictogramas
  • Historial de uso de pictogramas
  • Configuración de perfiles
  • Preferencias de la aplicación
  • Estadísticas de uso de pictogramas

Estos datos se almacenan únicamente en el dispositivo del usuario y no se transmiten a servidores externos.

2. Perfiles de usuario

La aplicación permite crear diferentes perfiles para adaptar la comunicación a distintas personas usuarias. Cada perfil guarda su propio historial de uso, configuraciones y preferencias.

Toda esta información se almacena únicamente en el dispositivo del usuario.

3. Fotos personalizadas

PictoTEA permite crear pictogramas personalizados utilizando fotografías tomadas con la cámara o seleccionadas desde la galería del dispositivo.

Estas imágenes se almacenan exclusivamente en el dispositivo del usuario y no se envían a servidores externos.

4. Estadísticas de uso

La aplicación puede generar estadísticas sobre el uso de pictogramas con el objetivo de ayudar a comprender cómo se utiliza la aplicación y mejorar la comunicación.

Estas estadísticas se calculan localmente en el dispositivo.

5. Uso opcional de inteligencia artificial

PictoTEA incluye una función opcional que permite generar una explicación automática del análisis de uso mediante un sistema de inteligencia artificial.

Cuando el usuario solicita esta función, se envía al servicio de IA un resumen del análisis de uso (frecuencia de pictogramas y patrones de comunicación).

Este resumen no incluye nombres, fotos ni datos personales identificables.

El objetivo de esta función es ofrecer una interpretación orientativa del uso de la aplicación que pueda ser útil para familias, docentes o profesionales.

6. Almacenamiento de datos

La mayor parte de los datos generados por la aplicación se almacenan únicamente en el dispositivo mediante almacenamiento local.

El usuario puede eliminar estos datos borrando perfiles o desinstalando la aplicación.

7. Compartición de datos

PictoTEA no vende, alquila ni comparte datos personales con terceros.

Los únicos datos que pueden enviarse a servicios externos son resúmenes anónimos utilizados para generar explicaciones automáticas mediante inteligencia artificial cuando el usuario activa explícitamente esta función.

8. Cambios en esta política

Esta política de privacidad puede actualizarse para reflejar mejoras o cambios en la aplicación. Las modificaciones se publicarán en esta misma página.
